# eda-cli

CLI-приложение на Python для Exploratory Data Analysis (EDA) CSV-файлов.  
Проект выполнен в рамках домашнего задания **HW03 (Seminar 03)**.

Инструмент позволяет получить краткий обзор датасета и сгенерировать EDA-отчёт
с таблицами, визуализациями и эвристиками качества данных.

## Подготовка окружения

Проект использует менеджер окружений **uv**.

# Установка зависимостей:
uv sync

# Команды CLI
Overview
Команда выводит краткий обзор датасета:
количество строк и столбцов;
типы данных;
число пропусков;
базовую статистику числовых признаков.
uv run eda-cli overview data/example.csv

# Базовый запуск:

uv run eda-cli report data/example.csv --out-dir reports_example
Параметры команды report
--title — заголовок отчёта (используется как # ... в report.md);

--min-missing-share — порог доли пропусков, выше которого колонка считается проблемной.

# Пример использования:

uv run eda-cli report data/example.csv \
  --out-dir reports_example \
  --title "HW03 EDA report" \
  --min-missing-share 0.3
# Эвристики качества данных
В проекте реализованы дополнительные эвристики качества данных, включая:
проверку наличия константных колонок;
выявление числовых колонок с высокой долей нулевых значений.

На основе эвристик рассчитывается интегральный показатель quality_score.

# Тесты
Проект содержит unit-тесты для проверки логики EDA и эвристик качества данных.

Запуск тестов:

uv run pytest -q
